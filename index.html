PROJECT_01
CLAIMS PLATFORM
Internal workflow system for logistics claims management
Live demo GitHub
What I built

End-to-end claims management system with role-based access control, a state machine workflow (open, review, approval, resolved), SLA tracking, full audit trail, and an analytics dashboard.

Supports 7 industries and 21 logistics clients. Deployed on AWS EC2 with Docker and nginx. Auto-seeds on startup. Generates credit notes on resolved cases.

Claims used to live in email chains and took 2 to 3 weeks to resolve. After building an automated workflow, it dropped to 2 days.

What I built

Trained an XGBoost model on 1.4 million Mercari product listings. Input a product name, brand, category, condition, and description. Get back a predicted price and confidence range.

Mean absolute error of $13.03 on a dataset with an average price of $27. FastAPI backend, React frontend, deployed on AWS EC2.
